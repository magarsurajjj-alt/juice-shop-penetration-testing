# Web Application Penetration Testing Report

## Target: OWASP Juice Shop

---

## 📌 Project Overview

This project demonstrates a full **Web Application Penetration Testing (WAPT)** assessment performed on OWASP Juice Shop, an intentionally vulnerable application.

The goal is to simulate real-world web attacks, identify vulnerabilities, exploit them, and provide mitigation strategies.

---

## 🎯 Objectives

* Identify security vulnerabilities
* Exploit common web application flaws
* Understand real-world attack scenarios
* Provide remediation techniques

---

## 🛠 Tools Used

* Burp Suite
* Browser DevTools
* curl / Postman
* Manual Testing

---

## 🧠 Methodology

1. Reconnaissance
2. Application Mapping
3. Vulnerability Identification
4. Exploitation
5. Reporting

---

## 🔥 Vulnerabilities Covered

* Cross-Site Scripting (XSS)
* SQL Injection (SQLi)
* Broken Access Control
* IDOR (Insecure Direct Object Reference)
* CSRF
* Authentication Issues
